# Dashboard de Vendas - Qlik Sense

Este repositório contém o script de ETL e o arquivo QVF do dashboard de vendas
desenvolvido no Qlik Sense Desktop.

## Conteúdo
- `script/` → script de carga em formato `.qvs`
- `qvf/` → versão compilada do app `.qvf`
- `datasets/` → base de dados de exemplo (opcional)

## Tecnologias
- Qlik Sense Desktop
- Qlik Script (QVS)

## Como abrir
Basta importar o QVF em qualquer Qlik Sense Desktop (trial ou licenciado).
