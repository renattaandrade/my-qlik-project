# My Qlik Project

A simple starter project for Qlik Sense apps and visualizations.

## Overview

This repository contains assets and examples for building Qlik Sense applications, including sample data and scripts to get you started.

## Prerequisites

- Qlik Sense (Desktop or Enterprise) or Qlik Cloud
- Basic familiarity with Qlik scripting and data modeling

## Getting Started

1. Clone the repository:

   git clone https://github.com/renattaandrade/my-qlik-project.git
2. Open Qlik Sense and import or load the provided data files or scripts.
3. Explore the example sheets and modify scripts as needed.
